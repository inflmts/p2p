# P2P File Sharing Project

This is the course project for CNT4007 Computer Network Fundamentals
at the University of Florida for the Fall 2025 semester.
Details about the project are available in
[description.pdf](description.pdf),
[logistics.pdf](logistics.pdf), and
[grading.pdf](grading.pdf).
We are tasked with writing a P2P file sharing software similar to BitTorrent.

## Compile

The executable is named `peer` (`peer.exe` on Windows).

```
make
```
